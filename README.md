## üåê Deployment

### GitHub Pages
1. Create a new repository on GitHub (e.g., `dhara-patel-portfolio`)
2. Upload your portfolio files
3. Go to Settings > Pages
4. Select source branch (usually `main`)
5. Your portfolio will be available at `https://dhara-patel.github.io/portfolio`

### Netlify
1. Drag and drop your portfolio folder to [Netlify](https://netlify.com)
2. Your site will be deployed instantly
3. Customize the URL in the site settings

### Vercel
1. Connect your GitHub repository to [Vercel](https://vercel.com)
2. Deploy automatically on every push
3. Get a custom domain and SSL certificate
